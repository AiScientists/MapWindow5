Priority list

In the last newsletter we asked to fill in a short survey about the development of MapWindow v5.
82 people filled it in. From the results the priority list below is compiled.

We will use this priority list as a guideline during the development. It helps us prioritize the many tasks we need to do.

If you want to keep informed about the progress of the development of MapWindow v5 you can read the posts at Codeplex: https://mapwindow5.codeplex.com/documentation . On a regular bases (we'll try every week) we will post our progress. This will include questions we have or issues we bump in to. Feel free to help with particular parts.

We've moved MapWinGIS from SVN to GIT: https://mapwingis.codeplex.com/SourceControl/latest . This makes it much easier for you to contribute to the project. Just fork the repository, make your changes and send us a pull request. More about the forking workflow can be read in this great tutorial: https://www.atlassian.com/git/tutorials/comparing-workflows/forking-workflow/

To make new MapWindow 5 GIS possible we need to raise $6,000. We already raised $4,500 (status end Februari 2015) so we've started with the developement.
It would be great to get the remaining funding. If you haven't donated yet, please do so. Every dollar helps.
