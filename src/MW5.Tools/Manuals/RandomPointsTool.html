Random points tool

If you need randomly generated point within a certain bounding box you can use this tool. Those points can, for example, be used to survey.

First determine the bounding box by selecting a layer.
Next select the number of points that should be created.
The output layer name is generated based on the input layer, but can be modified.
Check 'Overwrite' if you want to overwrite existing files. You'll be prompted otherwise if the output file already exists.
Check 'Memory layer' if you don't wan't to save the file to disk, but keep it only in memory.
'Add to map' is checked by default and will add the result to the map.
In the 'Tasks' panel you can see the progress of this task.
